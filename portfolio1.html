Projeto PHP sobre Controle de Estoque


Descrição do Projeto

Desenvolvi este sistema de controle de estoque durante o curso Técnico em Informática para Internet, utilizando PHP e MySQL. No projeto, implementei a lógica completa do CRUD (Create, Read, Update, Delete), permitindo o cadastro, listagem, edição e exclusão de produtos. Utilizei estrutura MVC, conexão com banco via PDO, formulários com validação e interface responsiva com Bootstrap. O foco foi aplicar boas práticas de organização de código e segurança nas requisições.
